# Sistema para reporte de sinistros de trânsito – Back-end (API ASP.NET Core) 
Este projeto corresponde ao back-end da aplicação InfoTrânsito, desenvolvido como uma API REST em ASP.NET Core, responsável por toda a validação, processamento e persistência dos dados relacionados aos sinistros de trânsito.
 
A API implementa a lógica de negócio, realiza a comunicação com o banco de dados MySQL e fornece os endpoints consumidos pelo front-end em React. O sistema contempla o gerenciamento completo de sinistros, veículos, pessoas envolvidas e demais entidades associadas, garantindo integridade, consistência e segurança das informações.

O desenvolvimento segue em evolução contínua, com foco na ampliação das regras de negócio, melhorias de validação, organização da arquitetura e integração com novas funcionalidades conforme a evolução da aplicação como um todo.
